# 新文档写作框架指南

## 文档分类框架

### 道层（3类）

1. **框架类**：思维框架和方法论
2. **理论类**：认知理论和AI理论
3. **哲学类**：人生哲学和价值观

### 术层（6类）

1. **创新类**：创新与创业方法论
2. **战略类**：战略思维与决策框架
3. **执行类**：执行技巧与实践方法
4. **沟通类**：沟通与社交技巧
5. **心理学类**：认知心理学与行为改变
6. **行为改变类**：行为改变方法

## 7步写作法

### 第1步：明确目标

- 文档要解决什么问题？
- 目标读者是谁？
- 期望达到什么效果？

### 第2步：确定结构

- 使用什么框架？
- 章节如何组织？
- 逻辑如何推进？

### 第3步：收集素材

- 相关文档有哪些？
- 案例和实例有哪些？
- 参考资源有哪些？

### 第4步：撰写初稿

- 按照结构撰写内容
- 注意语言表达
- 保持逻辑清晰

### 第5步：检查质量

- 内容是否完整？
- 逻辑是否清晰？
- 表达是否准确？

### 第6步：创建索引

- 生成YAML索引文件
- 添加关键词
- 完善摘要

### 第7步：提交审核

- 提交到仓库
- 等待审核
- 根据反馈修改

## 质量检查清单

- [ ] 文档分类正确
- [ ] 结构清晰完整
- [ ] 内容准确有价值
- [ ] 语言表达流畅
- [ ] 关键词和摘要完善
- [ ] YAML索引已创建
