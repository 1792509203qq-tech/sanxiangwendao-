# 思考的模型：定向 → 构建 (加法) → 聚焦 (减法) → 迭代 (增长与反馈)

> 思考的模型：定向 → 构建 (加法) → 聚焦 (减法) → 迭代 (增长与反馈)

## 核心观点

思考的模型提供了通用思维框架，适用于所有领域。

### 四个阶段

1. **定向**：明确方向和目标
2. **构建（加法）**：构建和添加要素
3. **聚焦（减法）**：聚焦和精简要素
4. **迭代（增长与反馈）**：迭代和持续改进

### 应用方法

- **定向阶段**：明确方向和目标
- **构建阶段**：构建和添加要素
- **聚焦阶段**：聚焦和精简要素
- **迭代阶段**：迭代和持续改进
