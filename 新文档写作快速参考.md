# 新文档写作快速参考

## 快速决策树

1. **文档类型**：道层 or 术层？
2. **内容分类**：框架/理论/哲学/创新/战略/沟通/心理学/行为改变？
3. **写作模式**：新文档写作模式

## 7步写作法模板

1. **明确目标**：文档要解决什么问题？
2. **确定结构**：使用什么框架？
3. **收集素材**：相关文档和案例
4. **撰写初稿**：按照结构撰写
5. **检查质量**：是否符合质量标准？
6. **创建索引**：生成YAML索引文件
7. **提交审核**：提交到仓库

## YAML索引模板

```yaml
doc_id: DOC-D001
title: 文档标题
category: 道层/术层
subcategory: 具体分类
keywords:
  - 关键词1
  - 关键词2
summary: 文档摘要
```
