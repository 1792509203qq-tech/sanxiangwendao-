本仓库已建立完整的RAG检索索引系统，支持智能问答和团队培训。

### 目录结构

```
rag-index/
├── config.yaml          # 总配置
├── template.yaml        # 索引模板
├── doc-mapping.md       # 文档ID映射（53篇文档）
├── learning-paths.md    # 学习路径
└── indexes/             # 各文档索引（53篇已建）
    ├── DOC-D001.yaml ~ DOC-D012.yaml  # 道层12篇
    └── DOC-S001.yaml ~ DOC-S041.yaml  # 术层41篇

rag-system/
├── config.py           # 配置文件
├── embedding.py        # Embedding模型封装
├── document_loader.py  # 文档加载器
├── vector_store.py     # 向量数据库封装
├── retriever.py        # 混合检索器
├── query_processor.py  # 查询处理器
└── rag_chain.py        # RAG链实现

scripts/
├── build_index.py      # 构建完整索引
├── update_index.py     # 增量更新索引
└── test_query.py       # 测试查询

```
