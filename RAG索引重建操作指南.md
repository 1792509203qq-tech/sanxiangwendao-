# RAG索引重建操作指南

## 概述

本文档说明如何重建RAG索引系统。

## 操作步骤

### 1. 备份现有索引

在重建之前，建议先备份现有的索引文件。

### 2. 清理旧索引

删除 `rag-index/indexes/` 目录下的所有旧索引文件。

### 3. 运行重建脚本

```bash
python scripts/build_index.py
```

### 4. 验证索引

运行测试脚本验证索引是否正确：

```bash
python scripts/test_query.py
```

## 注意事项

- 重建索引可能需要较长时间
- 确保所有文档文件都已准备好
- 检查配置文件是否正确
